# GCashMall

This is an online store app, with multi-language support.

Follow the pages/API specs under the specs folder and generate the corresponding code:

- react components
- data models
- API calls
- seed data in db

This app supports Android, use Apache Cordova to package the app. 

The display name of the app is defined as APP_DISPLAY_NAME in .env file. Any appearance of "GcashTV" or "GCashMall" in any UI page should be replaced with APP_DISPLAY_NAME.

## Accounts

### User

A regular user who can watch/purchase videos under his/her own account.

### Creator

A regular user who has access to My Series, video upload, video creation and revenue sections.

### Admin

A creator who has access to system settings.
