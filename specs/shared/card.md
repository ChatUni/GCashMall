# Card component

A container component.

## Style
Poster Image
- Aspect ratio: vertical (2:3)
- rounded corner: 4px
- bg color: black / near-black  (#0B0B0E)
- Image fully covers the container

Title
- Font size: small–medium
- Color: white
- Maximum lines: 1–2 with ellipsis
- Small margin above the title for spacing

## Layout
- Vertical
- poster image
- title text
- tag (only one, the most representative one)

## Interaction
- On hover: Poster image slightly scales up，subtle glows and slightly scales up, scale up by 2%, animate the change in 0.3 sec
- On click: Navigate to the player page
