# Phone About Page Specification

## Overview

The Phone About page displays information about the GCashTV application, including app details, features, and legal information.

## Page Structure

### Layout
- Uses the standard phone layout
- Back button and "About" title in header
- No search icon
- Bottom navigation visible

### Content Sections
1. App Logo and Version
2. About Description
3. Features List
4. Legal Links
5. Copyright Footer

## App Logo Section

### Container
- Centered content
- Generous vertical padding
- Optional subtle background

### Logo
- GCashTV logo image
- 64 pixel height

### App Name
- "GCashTV" text
- Bold, white, 24 pixel font

### Version
- Version number
- Gray text, 14 pixel font

## About Description

### Container
- Centered text
- Horizontal padding

### Text
- Brief description of the app
- Gray text, readable line height
- Explains what GCashTV offers

## Features Section

### Container
- Rounded card background
- Horizontal margin

### Section Title
- "Features" heading
- Bold, white, 16 pixel font

### Feature Items
Each feature shows:
- Icon on left (blue, 24 pixels)
- Description text (white, 14 pixels)
- Subtle border between items

### Listed Features
- HD Video Streaming
- Multi-language Support
- Watch History Sync
- Offline Downloads
- Secure and Private

## Legal Links Section

### Container
- Vertical list of links
- Horizontal padding

### Link Items
Each link shows:
- Link text on left (white)
- Arrow icon on right (gray)
- Rounded background
- Tappable full width

### Available Links
- Terms of Service
- Privacy Policy
- Open Source Licenses

## Copyright Footer

### Container
- Centered at bottom
- Padding above and below

### Text
- Copyright notice
- Gray text, small font
- Example: "© 2024 GCashTV. All rights reserved."

## Interactions

| Element | Action | Result |
|---------|--------|--------|
| Back Button | Tap | Return to previous page |
| Terms of Service | Tap | Open terms page |
| Privacy Policy | Tap | Open privacy page |
| Licenses | Tap | Open licenses page |

## Internationalization

### Labels
- English: "About", "Version", "Features", "Terms of Service", "Privacy Policy", "Open Source Licenses"
- Chinese: "关于我们", "版本", "功能特点", "服务条款", "隐私政策", "开源许可"

### Feature Descriptions
Feature text is translated based on selected language.
